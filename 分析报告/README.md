# 分析报告与冲分规划

本目录存放针对 CCL2026-MGBIE 比赛的深度数据分析报告、得分诊断以及分阶段的冲分规划。

## 核心报告

### 1. 赛道与数据偏差分析
- `1_赛道与数据偏差分析.md`：详细分析了 AI 修正训练集导致的严重标准偏移风险（如 AFF 严重过标、LOI 双向混乱），并建议放弃修正数据，直接使用官方原始标注。

### 2. 重构方案与官方规律
- `2_重构方案与官方规律.md`：基于官方 1000 条训练数据的真实统计规律，提出了"原汁原味"的重构方案，明确了 LOI 和 AFF 的真实分布，并制定了 Track-A 的预测策略。

### 3. 技能重构与预测逻辑
- `3_技能重构与预测逻辑.md`：解释了为什么将 `mgbie-ner-annotator` 从"数据清洗工具"彻底改写为"A 榜预测工具"，并说明了预测机制的本质。

### 4. 冲分规划（突破 0.350）
- `4_冲分规划_突破0.350.md`：基于 v3 预测结果（0.3482 分）的深度诊断，量化了每个失分点的分值贡献，并制定了三阶段的冲分路线图。
