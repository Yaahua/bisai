## 冲 0.5 路径评估

当前 0.4487 → 0.5 需要 **+0.0513**，这是一个较大跨越。

| 路径 | 理论上限 | 可行性 |
| :--- | :--- | :--- |
| 继续加法（safe 候选池） | +0.001~0.002 | 低（池子耗尽） |
| 减法策略（删假阳性） | +0.005~0.015 | 中（需精准定位） |
| LLM 有关系样本补充 | +0.005~0.010 | 中（LLM 精度约 60%） |
| 全新 LLM 预测（替换底座） | +0.020~0.050 | 高风险（可能大幅掉分） |

**结论**：单靠加法无法冲到 0.5，需要减法+加法组合，或者引入更强的 LLM（如 GPT-4o）对全量 test_A 重新预测。
