# 杂粮育种外部知识库 (Breeding Knowledge Base)

本知识库为 `mgbie-review-agent` 提供对抗性审查所需的专业领域知识，涵盖谷子、高粱、燕麦、荞麦等杂粮作物的术语规范、基因命名规则及常见易混淆概念。

## 1. 基因与 QTL 命名规范

在进行**知识审查**时，需严格核对实体是否符合以下国际与国家标准命名规则：

### 1.1 基因 (GENE) 命名规则
- **全称与符号**：基因符号通常由 2-4 个英文字母组成，首字母大写或全大写（如 *Waxy*, *ABI5*）。
- **突变体**：隐性突变基因通常全小写（如 *sd1*），显性基因首字母大写（如 *Sd1*）。
- **易混淆点**：基因名称与性状名称极易混淆。例如“抗旱基因”，其中“抗旱”是性状 (TRT)，“基因”是实体 (GENE)。审查时需确保分类粒度最细。

### 1.2 数量性状位点 (QTL) 命名规则
- **标准格式**：通常以小写字母 `q` 开头，后接性状缩写和染色体编号。
- **示例**：`qPH3.1` (q + Plant Height + 染色体3 + 序号1)，`qYLD2.2` (q + Yield + 染色体2 + 序号2)。
- **审查要点**：带有 `q` 前缀且包含数字编号的通常为 QTL，而非普通基因或分子标记。

### 1.3 分子标记 (MRK) 命名规则
- **常见类型**：SSR (Simple Sequence Repeat), SNP (Single Nucleotide Polymorphism), InDel, KASP。
- **命名特征**：通常包含标记类型前缀或实验室代号，如 `SSR-Xgwm20`, `RM190`, `KASP_Chr2_15Mb`。
- **审查要点**：区分标记 (MRK) 与其定位的 QTL/基因。标记是用于检测的工具，QTL/基因是实际的遗传位点。

## 2. 农艺性状 (TRT) 分类体系

在进行**语义审查**和**词块审查**时，需参考以下性状分类，确保没有遗漏或错误归类：

- **形态特征**：株高、穗长、茎粗、叶面积、分蘖数。
- **产量构成**：千粒重、穗粒数、单株产量、结实率。
- **品质特性**：蛋白质含量、淀粉含量、直链淀粉/支链淀粉比例、赖氨酸含量。
- **抗逆性 (与胁迫相关)**：抗旱性、抗寒性、耐盐碱性、抗倒伏性、抗病性（如抗黑穗病）。
- **生育期**：全生育期天数、抽穗期天数。

## 3. 常见作物与品种 (CROP vs VAR) 辨析

- **作物 (CROP)**：指物种大类，如谷子 (Foxtail millet)、高粱 (Sorghum)、燕麦 (Oat)、荞麦 (Buckwheat)、糜子 (Proso millet)。
- **品种 (VAR)**：指具体培育出的商业化或实验化品系，通常带有数字、地名或特定代号。
  - **示例**：晋谷21（谷子品种）、晋燕8号（燕麦品种）、中绿5号（绿豆品种）。
- **审查要点**：文本中若出现“晋谷21等谷子品种”，需明确拆分为 `晋谷21` (VAR) 和 `谷子` (CROP)，并建立 `CONTAINS` 关系。

## 4. 胁迫类型 (ABS vs BIS) 辨析

- **非生物胁迫 (ABS)**：环境因素。如干旱 (Drought)、高温 (Heat)、低温/冷害 (Cold/Chilling)、盐碱 (Salinity)、低氮 (Low Nitrogen, LN)。
- **生物胁迫 (BIS)**：病虫害因素。如蚜虫 (Aphid)、螟虫 (Borer)、白粉病 (Powdery mildew)、黑穗病 (Smut)。
- **审查要点**：严格区分病害（生物）与环境（非生物）造成的胁迫。

## 5. 审查原则与对抗性策略

1. **拆分审查**：检查长复合词是否被错误地作为一个整体实体。例如“高粱抗旱性相关基因”应拆分为 CROP, TRT, GENE 三个实体。
2. **语义审查**：检查关系 (RE) 的方向和逻辑是否合理。例如 `AFFECTS` 关系通常是胁迫/基因影响性状，方向不能反。
3. **词块审查**：检查实体边界是否包含多余的修饰词（如“严重的”、“显著的”）。
4. **知识审查**：利用本知识库中的命名规则，核对 QTL、基因、标记的分类是否准确。
