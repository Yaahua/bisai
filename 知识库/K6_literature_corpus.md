# 杂粮育种信息抽取：文献语料库与标注示例 (K6)

本文档基于高引用农业文献和 CCL 2026 MGBIE 任务的官方数据结构，提供杂粮育种领域的典型句子及其标准标注示例，作为模型微调和 Prompt 提示的 Few-shot 样本库。

## 1. 谷子 (Foxtail Millet) 语料

### 1.1 基因与性状调控
**原文**：*SiLEA14* 基因的过表达显著提高了转基因谷子的抗旱性和耐盐性，同时增加了干旱胁迫下的相对含水量和脯氨酸含量。
**实体标注**：
- `Gene`: [SiLEA14基因]
- `Agronomic_Trait`: [抗旱性], [耐盐性], [相对含水量], [脯氨酸含量]
- `Environment`: [干旱胁迫]
**关系标注**：
- `Regulate`: [SiLEA14基因] -> [抗旱性]
- `Regulate`: [SiLEA14基因] -> [耐盐性]
- `Regulate`: [SiLEA14基因] -> [相对含水量]
- `Regulate`: [SiLEA14基因] -> [脯氨酸含量]
- `Regulate`: [干旱胁迫] -> [相对含水量]
- `Regulate`: [干旱胁迫] -> [脯氨酸含量]

### 1.2 品种与农艺性状
**原文**：晋谷21号是一个优质的夏谷品种，其突出特点是出谷率高、黄小米色泽鲜艳，且对谷锈病表现出中抗水平。
**实体标注**：
- `Variety`: [晋谷21号]
- `Agronomic_Trait`: [出谷率], [黄小米色泽], [谷锈病抗性] (注：根据最大共识原则，"中抗水平"是状态，不包含在实体内)
**关系标注**：
- `Attribute`: [晋谷21号] -> [出谷率]
- `Attribute`: [晋谷21号] -> [黄小米色泽]
- `Attribute`: [晋谷21号] -> [谷锈病抗性]

## 2. 燕麦 (Oat) 语料

### 2.1 QTL 定位
**原文**：利用坝莜3号和坝莜6号构建的 RIL 群体，在第 7 连锁群上定位到一个控制燕麦皮裸性的主效 QTL *qHN-7.1*，解释了 45.2% 的表型变异。
**实体标注**：
- `Variety`: [坝莜3号], [坝莜6号]
- `QTL`: [qHN-7.1]
- `Agronomic_Trait`: [皮裸性]
**关系标注**：
- `Locate`: [qHN-7.1] -> [皮裸性]

### 2.2 标记开发
**原文**：SSR 标记 Xgwm261 与燕麦的穗下节茸毛性状紧密连锁，遗传距离为 2.5 cM，可用于该性状的分子标记辅助选择。
**实体标注**：
- `Molecular_Marker`: [SSR标记Xgwm261]
- `Agronomic_Trait`: [穗下节茸毛]
**关系标注**：
- `Associate`: [SSR标记Xgwm261] -> [穗下节茸毛]

## 3. 高粱 (Sorghum) 语料

### 3.1 基因家族分析
**原文**：全基因组鉴定表明，高粱 *SbWRKY* 基因家族包含 72 个成员，其中 *SbWRKY45* 在高盐胁迫下表达量急剧上调，可能参与了耐盐性调控。
**实体标注**：
- `Gene`: [SbWRKY基因家族], [SbWRKY45]
- `Environment`: [高盐胁迫]
- `Agronomic_Trait`: [耐盐性]
**关系标注**：
- `Regulate`: [高盐胁迫] -> [SbWRKY45] (表达量上调)
- `Regulate`: [SbWRKY45] -> [耐盐性]

### 3.2 品种比较试验
**原文**：在 2023 年的区域试验中，辽杂 19 号的平均株高为 165 cm，比对照品种晋杂 22 号矮 15 cm，表现出极强的抗倒伏能力。
**实体标注**：
- `Variety`: [辽杂19号], [晋杂22号]
- `Agronomic_Trait`: [株高], [抗倒伏能力]
**关系标注**：
- `Attribute`: [辽杂19号] -> [株高]
- `Attribute`: [晋杂22号] -> [株高]
- `Attribute`: [辽杂19号] -> [抗倒伏能力]

## 4. 复杂嵌套与共指语料

### 4.1 嵌套实体处理
**原文**：*qPH-1* 区域内包含一个候选基因 *SiGA2ox*，该基因编码赤霉素 2-氧化酶，通过降低活性赤霉素含量来控制谷子的矮秆表型。
**实体标注**：
- `QTL`: [qPH-1]
- `Gene`: [SiGA2ox]
- `Agronomic_Trait`: [赤霉素含量], [矮秆表型] (注：不标注酶的名称，因为任务重点是农艺性状)
**关系标注**：
- `Locate`: [qPH-1] -> [SiGA2ox] (包含关系，可映射为 Associate)
- `Regulate`: [SiGA2ox] -> [赤霉素含量]
- `Regulate`: [SiGA2ox] -> [矮秆表型]
