# 杂粮育种权威引用库 (Authoritative References)

本引用库汇集了杂粮育种领域的国家标准、农业行业标准、权威专著及顶尖学术论文。
**核心要求**：在进行分类（NER/RE）和审查时，**必须**从本库中寻找依据，并在输出的分类说明文档或审查文档中明确标注引用来源（如：`依据 [NY/T 2645-2014]，该性状属于...`）。

## 4. 引用要求与执行逻辑

1. **强制引用**：在遇到易混淆实体（如 GENE vs TRT, ABS vs BIS, CROP vs VAR）时，必须在说明文档中引用上述编号。
2. **格式规范**：引用格式必须为 `依据 [编号]，...`。
3. **无明确出处时**：若实体过于生僻未在上述列表中，需引用 `[通用生物学常识]` 或 `[上下文推断]`，并给出详细的逻辑推导过程。
