## 关键经验总结

### 有效策略
- **多模型集成（三选二）**：v7 + v9 + v10 的三重集成是核心高分来源，精确率极高。
- **极保守白名单规则**：基于训练集触发词分析，只补充"几乎不可能是 FP"的关系（如 `QTL on chromosome`），在不损失精确率的前提下提升召回率。

### 失败策略
- **"加法"集成（v20 系列）**：在弱模型集成的基础上引入自一致性采样（SC）和白名单扩展，新增关系的精确率极低（约 1%），大量假阳性涌入导致 F1 增益几乎为零甚至为负。
- **大模型 NLI 验证器（v21/v22）**：GLM-4-Flash 对所有关系给出 1.0 置信度，批量盖章，无真实区分能力。
- **大模型作为第四投票模型（v24/v25/v26）**：DeepSeek 单体关系均值高达 3.99，严重过标，与现有模型重叠度低。
- **泛化规则后处理（v16）**：基于 `of`、`for` 等通用介词的规则过宽，引入大量 FP，精确率崩溃。
