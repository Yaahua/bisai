# 官方原始数据

本目录用于单独保存**来自比赛官方仓库的原始训练数据**，与仓库中用于实验、提交构造和结果产物的其他数据文件分开管理，避免混淆。

## 当前内容

| 文件 | 说明 |
| :--- | :--- |
| `train.json` | 官方原始训练集，未做任何 AI 修订、清洗或重写。 |

## 设计原则

本目录强调“**原样保留**”。也就是说，这里的 `train.json` 应始终被视为官方基准数据，后续如果需要做统计分析、Few-shot 检索、规则提取或误差诊断，应当基于该文件读取，但不应直接在这里改写内容。

为了避免历史上出现过的“AI 按学术标准修正官方标注、导致训练标准偏移”的问题，本目录专门独立出来，目的就是把**官方原始标注**和**项目过程产物**彻底隔离。

## 与其他目录的关系

| 目录 | 作用 | 与本目录关系 |
| :--- | :--- | :--- |
| `数据/训练集/` | 项目运行时已接入的数据位置 | 可用于脚本兼容，但不强调“原始只读”语义 |
| `数据/A榜/` | 各版本提交 JSON / ZIP | 与官方原始训练数据分离 |
| `分析报告/` | 基于训练集做出的统计和诊断文档 | 其结论应可回溯到本目录的原始文件 |
