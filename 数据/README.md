# 数据目录

本目录存放比赛相关的数据文件，包括官方原始训练数据、官方原始 A 榜测试数据、项目运行时接入的数据文件以及 A 榜预测结果。

## 目录结构

| 子目录 | 用途说明 |
|------|------|
| `A榜/` | 存放历次生成的预测结果 JSON 文件及打包好的 ZIP 文件，可直接用于天池提交。 |
| `训练集/` | 项目当前脚本直接读取的训练数据位置，用于兼容历史脚本与分析流程。 |
| `官方原始数据/` | 单独保存来自官方仓库的原始 `train.json` 与 `test_A.json`，强调原样保留、只读基准和可追溯性。 |

## 管理原则

`官方原始数据/` 与 `训练集/`、`A榜/` 的语义不同。前者是为了在仓库中明确保留一份**不经 AI 修订、不做人工重写**的官方基准文件；后两者则分别服务于脚本兼容与提交产物管理。因此，在需要追溯“最原始官方数据”时，应优先参考 `官方原始数据/train.json` 与 `官方原始数据/test_A.json`。

> **注意**：本方案已放弃对训练集进行 AI 修正，所有预测、统计和规则提取都应以官方原始标注规律为准。历史上任何“纠正式”数据都不应再作为基准使用。
